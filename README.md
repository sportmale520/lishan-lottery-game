# 梨山國中小 ‧ 臺灣文學行動展主題抽獎系統

本專案是為 **臺中市立梨山國民中小學** 在舉辦「臺灣文學行動展」期間所開發的數位主題抽獎系統。主要針對原住民文化與文學相關脈絡，為繳交觀後學習單的學生提供充滿儀式感的數位抽獎體驗。

👉 **線上即時體驗網址**：https://sportmale520.github.io/lishan-lottery-game/

---

## 🎨 系統設計特色

### 1. 泰雅族文化美學
- **傳統配色**：視覺採用泰雅族傳統三色（朱紅、黑、白）為主色調，營造尊榮且溫暖的氛圍。
- **祖靈之眼 (Dowriq Utux)**：Canvas 畫布背景融入象徵保護與誠實的菱形紋路水印。
- **高山營火意象**：結合梨山高山雲霧與溫暖營火粒子特效，象徵文學傳承之火不熄。

### 2. 雙主題互動抽獎儀式 (HTML5 Canvas)
- **🎯 泰雅神箭獵星 (Archery)**：射箭動畫。左側有傳統木雕弓與紅白色羽毛箭矢，右側候選名單如星環般圍繞著「祖靈之眼」靶心旋轉。點擊後拉弓射出，擊中靶心爆發星火粒子。
- **🌀 織路祖靈轉輪 (Weaving)**：編織轉輪。圓輪印有精美泰雅編織幾何紋路，旋轉減速指向得獎者，並爆發菱形發光波紋。

### 3. 語音公告與音效系統
- **教務處領獎提示**：介面設有顯眼的「教務處教學組重要通知」，引導中獎學生至教務處教學組領取獎品。
- **中文唱名廣播**：調用 Web Speech API，在中獎時自動用語音念出：`「恭喜 [班級] [姓名] 同學中獎，獲得 [獎品]！請至教務處教學組領取獎品！」`，非常適合抽獎現場廣播。
- **即時合成音效**：使用 Web Audio API 即時運算並播放拉弦、射箭「咻」聲及中獎和弦聲，免除加載外部音檔失敗的風險。

## 📂 專案檔案結構
- `index.html` - 網頁主結構、設定面板與通知公告欄
- `style.css` - 泰雅主題配色、玻璃海報風與動畫關鍵影格
- `app.js` - 名單生命週期、Canvas 動態繪圖引擎、Web Audio 合成器與語音唱名邏輯
- `README.md` - 專案說明文件

## 💻 快速開始與使用方式

本系統為純前端單頁面應用 (Single Page Application)，無需安裝任何伺服器或資料庫：
1. 下載本專案所有檔案。
2. 直接雙擊 `index.html` 即可在瀏覽器中開啟使用！
3. 在左側名單區可直接黏貼或編輯您的學生名單（格式：`班級 姓名`，例如：`七年甲班 林建宏`），設定獎品名稱與抽取人數後，點擊「開始抽獎」即可。
4. 抽獎結束後，可點擊「匯出名單」下載 CSV 檔案以利紀錄。
